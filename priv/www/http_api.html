HTTP API

简介

除了这个帮助页面, 所有的URI提供的数据类型都是application/json, 并且都要求HTTP基本身份验证. 默认的用户名/密码是 admin/admin.

此emqttd_dashboard插件提供web管理控制台。仪表板帮助管理员监测broker（消息转发器）的运行状态，并对MQTT数据包进行统计和度量。

API列表

GET 路径 描述
X /api/stats 获取对客户端、会话、主题和订阅的统计
X /api/nodes 获取集群节点列表
X /api/metrics 获取broker（消息转发器）的度量
X /api/listeners 获取TCP监听端列表
X /api/clients 获取全部已连接的客户端列表
X /api/sessions 获取全部会话列表
X /api/topics 获取所有消息主题列表
X /api/routes 获取全部路由列表
X /api/subscriptions 获取所有消息订阅列表
X /api/users 获取管理员列表
